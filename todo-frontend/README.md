# todo-frontend

## Prerequisites
- Node.js (use the version compatible with `npm@11.6.2`)
- npm 11.6.2

## Install
```sh
npm install
```

## Run the dev server
```sh
npm run start
```
Open `http://localhost:4200/` in your browser.

## Build
```sh
npm run build
```

## Watch build (development)
```sh
npm run watch
```

## Test
```sh
npm run test
```
